# `scripts/` — 辅助脚本

| 脚本 | 说明 | 运行方式 |
|---|---|---|
| `generate-brand-assets.py` | 生成多平台图标资产（Windows/macOS/Linux） | `python scripts/generate-brand-assets.py` |
| `smoke-test.bat` | 冒烟测试：构建 → 启动 → 验证进程 → 退出 | `scripts\smoke-test.bat` |
| `package-release.ps1` | 打包发行版 | `powershell scripts\package-release.ps1` |
