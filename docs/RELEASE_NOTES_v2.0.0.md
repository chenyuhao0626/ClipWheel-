# ClipWheel v2.0.0

发布日期：2026-04-18

ClipWheel 2.0 是一次面向“可用性 + 视觉一致性 + 发布完整度”的大版本升级。这个版本重点优化了轮盘选择到粘贴的关键路径，补齐了控制中心管理能力，并建立了完整的应用图标与发布资产体系。

## 亮点更新
- 重构轮盘交互流程，优化“呼出 -> 选择 -> 松开”体验，提升选择反馈与可见性。
- 控制中心升级为主界面，支持固定项与历史项的统一管理、热键与自动粘贴配置。
- 新增并完善托盘体验：显示主窗口、退出应用、关于等常见入口，支持最小化到托盘。
- 全量升级品牌图标资产，提供多分辨率图标与跨平台资源（Windows/macOS/Linux）。
- 提供项目介绍网站（单文件页面），可直接用于 GitHub Pages。

## 改进细节
- 优化 UI 对比度、层级与排版，改善此前白色界面在复杂背景下可读性不足的问题。
- 调整组件布局与状态显示，减少控件重叠与视觉噪音。
- 增强应用资源打包链路，统一窗口图标、任务栏图标、托盘图标等视觉元素来源。

## 资源与发布
- 建议附件：`clipwheel.exe`、`clipwheel.ini`、`README.md`
- 可选附件：安装包（如 Inno Setup 输出）、演示截图/GIF

## 升级说明
- 推荐覆盖升级到 2.0.0，并保留已有 `clipwheel.ini` 配置。
- 如热键冲突，请在控制中心修改热键后保存。

## 已知说明
- 当前版本主要面向 Windows 桌面环境；macOS/Linux 图标资源已提供，用于后续跨平台发布准备。
