# ClipWheel v2.1.0

发布日期：2026-04-25

## 本次发布重点
- 主界面与轮盘整体视觉重构，统一为更简约、清晰、稳定的深色产品化风格。
- 修复控制中心闪烁、轮盘抖动和“鬼影”问题，保证交互稳定性优先。
- 鼠标自动回中逻辑升级为“当前显示器中心”，并与轮盘命中中心统一，提升选择灵敏度。
- 固定项删除逻辑优化：从“清空全部固定项”改为“删除选中固定项”，降低误操作风险。

## 改进明细
- Overlay/主界面/自绘列表重绘路径优化，减少不必要刷新引发的闪烁。
- 轮盘呼出与透明度过渡调整为更平滑、无抖动的动画曲线。
- 轮盘中心死区优化，提升从中心出发时的选区命中率。
- 托盘菜单与主界面按钮文案同步调整为“删除选中固定项”。
