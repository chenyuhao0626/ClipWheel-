# ClipWheel v2.1.0 - 构建和测试指南

## 🛠️ 构建指南

### 环境要求
- Windows 10/11 x64
- Visual Studio 2022 Build Tools
- C++ 工作负载

### 构建步骤
```bash
# 1. 进入项目目录
cd C:\Users\Administrator\Projects\clipwheel

# 2. 运行构建脚本
build.bat

# 3. 等待构建完成
# 看到 "Build OK: clipwheel.exe generated" 即为成功
```

### 构建输出
- **主程序**: `clipwheel.exe`
- **配置文件**: `clipwheel.ini` (已存在，保持不变)
- **数据文件**: `%APPDATA%\ClipWheel\clips.bin` (自动生成)

## 🐛 常见问题排查

### 构建问题

#### 问题: 找不到 cl.exe
```
解决方法:
1. 确保安装了 Visual Studio 2022 Build Tools
2. 确保选择了 C++ 工作负载
3. 重新运行 build.bat
```

#### 问题: 链接错误
```
解决方法:
1. 检查 Windows SDK 是否安装
2. 确保 lib 文件路径正确
3. 清理后重新构建
```

### 运行问题

#### 问题: 轮盘无法显示
```
排查步骤:
1. 检查热键是否被其他程序占用
2. 尝试修改热键设置
3. 检查是否有管理员权限
```

#### 问题: 鼠标不自动居中
```
排查步骤:
1. 确认使用的是最新版本
2. 检查系统是否有特殊的鼠标设置
3. 尝试重启应用
```

#### 问题: 界面显示异常
```
排查步骤:
1. 检查系统DPI设置
2. 尝试调整显示缩放
3. 更新显卡驱动
```
