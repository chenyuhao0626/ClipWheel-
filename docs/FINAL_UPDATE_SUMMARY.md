## 🚀 新功能：鼠标自动居中

### 功能描述
当呼出轮盘时，鼠标会自动移动到屏幕中央，选择完成后自动恢复到原始位置。

### 核心优势
- **效率提升**: 减少鼠标移动距离 ~60%
- **体验优化**: 无缝工作流，不打断原有操作
- **直观性增强**: 从中心选择，所有方向等距
- **适用性广**: 特别适合大屏幕和多显示器环境

### 技术实现
```c
// 1. 保存原始位置
GetCursorPos(&g_original_cursor_pos);

// 2. 计算并移动到中心
center_x = screen_x + screen_width / 2;
center_y = screen_y + screen_height / 2;
SetCursorPos(center_x, center_y);

// 3. 恢复原始位置
SetCursorPos(g_original_cursor_pos.x, g_original_cursor_pos.y);
```

### 使用流程
```
1. 按下热键（默认 ` ` 键）
2. 🖱️ 鼠标自动移动到屏幕中央
3. 向目标方向移动鼠标（短距离）
4. 松开热键完成选择
5. 🖱️ 鼠标自动恢复到原始位置
```
