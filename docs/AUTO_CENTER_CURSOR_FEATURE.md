# ClipWheel 新功能 - 鼠标自动居中

## 🎯 功能概述

当呼出轮盘时，鼠标会自动移动到屏幕中央（轮盘中心），完成后自动恢复到原始位置，大幅提升选择效率和用户体验。

## ✨ 功能亮点

### 🚀 选择效率提升
- **无需长距离移动**: 从任何位置都能快速到达轮盘中心
- **等距选择**: 所有扇区距离相等，选择更加直观
- **肌肉记忆优化**: 形成稳定的"热键 → 居中 → 方向选择"模式

### 🎪 用户体验优化
- **无缝工作流**: 鼠标自动恢复，不打断原有操作
- **视觉一致性**: 始终从中心开始选择，视觉体验统一
- **操作连贯性**: 无论确认还是取消，鼠标都会恢复

## 🔧 技术实现

### 核心逻辑
```c
// 1. 显示轮盘时
static void show_wheel(void) {
    // 保存当前鼠标位置
    GetCursorPos(&g_original_cursor_pos);

    // 计算屏幕中心
    center_x = vx + vw / 2;
    center_y = vy + vh / 2;

    // 移动鼠标到中心
    SetCursorPos(center_x, center_y);
}

// 2. 隐藏轮盘时（确认/取消）
static void hide_wheel_commit(void) {
    // 恢复鼠标到原始位置
    SetCursorPos(g_original_cursor_pos.x, g_original_cursor_pos.y);
}
```

### 关键变量
```c
static POINT g_original_cursor_pos; // 保存鼠标原始位置
```

## 📋 操作流程

### 正常选择流程
```
1. 用户在屏幕任意位置（比如编辑器右下角）
2. 按下热键呼出轮盘
3. 鼠标自动移动到屏幕中央
4. 用户向目标方向移动鼠标（短距离）
5. 松开热键确认选择
6. 鼠标自动恢复到编辑器右下角
7. 继续编辑工作
```

### 取消流程
```
1. 用户在屏幕任意位置
2. 按下热键呼出轮盘
3. 鼠标自动移动到屏幕中央
4. 用户按 ESC 取消
5. 鼠标自动恢复到原始位置
6. 继续原有工作
```

## 🎨 用户体验优势

### 传统方式 vs 新功能

#### 传统方式
```
原始位置: 屏幕右下角
↓ 呼出轮盘
需要移动: 大幅度移动到中央 + 再移动到目标扇区

```

```
↓ 呼出轮盘
自动移动: 瞬间到达中央 + 短距离移动到目标扇区
↓ 选择完成
自动恢复: 瞬间回到原始位置（右下角）
```

## 🔍 技术细节

### 鼠标位置保存
- **时机**: 轮盘显示前立即保存
- **精度**: 使用 POINT 结构保存精确坐标
- **范围**: 支持多显示器虚拟屏幕

### 屏幕中心计算
```c
vx = GetSystemMetrics(SM_XVIRTUALSCREEN);  // 虚拟屏幕左上角X
vy = GetSystemMetrics(SM_YVIRTUALSCREEN);  // 虚拟屏幕左上角Y
vw = GetSystemMetrics(SM_CXVIRTUALSCREEN);  // 虚拟屏幕宽度
vh = GetSystemMetrics(SM_CYVIRTUALSCREEN);  // 虚拟屏幕高度

center_x = vx + vw / 2;  // 中心X坐标
center_y = vy + vh / 2;  // 中心Y坐标
```

### 位置恢复时机
- **确认选择**: `hide_wheel_commit()` 函数中
- **取消操作**: `hide_wheel_cancel()` 函数中
- **一致性**: 两种情况都会恢复，体验统一
