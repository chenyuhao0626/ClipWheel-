# 🎯 ClipWheel 新功能：鼠标自动居中

## 📋 功能描述

当您按下热键呼出轮盘时，鼠标会自动移动到屏幕中央，选择完成后自动恢复到原始位置。

## ✨ 核心优势

### 🚀 效率提升
- **减少移动距离**: 从屏幕角落到中央的移动由系统自动完成
- **等距选择**: 从中央出发，所有扇区距离相等
- **快速恢复**: 完成后鼠标瞬间回到原位，不打断工作流

### 🎯 特别适用
- 🖥️ 大屏幕显示器 (34寸+)
- 🖥️🖥️ 多显示器配置
- ⚡ 高频剪贴板操作
- 🎯 需要保持鼠标位置的精确编辑

## 🎮 使用方法

### 正常选择
```
1. 按下热键（默认 ` ` 键）
2. 🖱️ 鼠标自动移动到屏幕中央
3. 向目标方向移动鼠标（短距离）
4. 松开热键完成选择
5. 🖱️ 鼠标自动恢复到原始位置
```

### 取消操作
```
1. 按下热键呼出轮盘
2. 🖱️ 鼠标自动移动到屏幕中央
3. 按 ESC 取消
4. 🖱️ 鼠标自动恢复到原始位置
```

## 📊 效果对比

### 传统方式
```
原始位置 → 长距离移动到中央 → 移动到扇区 → 选择 → 鼠标停留
```

### 新功能

## 🔧 技术细节

### 核心实现
```c
// 保存原始位置
GetCursorPos(&g_original_cursor_pos);

// 移动到中央
center_x = screen_x + screen_width / 2;
center_y = screen_y + screen_height / 2;
SetCursorPos(center_x, center_y);

// 恢复原始位置
SetCursorPos(g_original_cursor_pos.x, g_original_cursor_pos.y);
```
